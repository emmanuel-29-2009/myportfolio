# Modern Portfolio with 3D Animations

A professional portfolio website featuring stunning 3D animations built with React Three Fiber.

## Features

- 🎨 Modern, professional design
- 🚀 3D animations with React Three Fiber
- 💫 Smooth scroll animations with Framer Motion
- 📱 Fully responsive
- 🎯 3 project slots as requested
- 📧 Contact form
- ⚡ Fast performance

## Technologies Used

- React.js
- React Three Fiber (3D)
- Framer Motion (Animations)
- CSS3
